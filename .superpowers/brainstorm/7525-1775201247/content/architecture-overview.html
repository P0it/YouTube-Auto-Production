게임 루프

Flame의 게임 루프가 카드 물리, 애니메이션 타이밍, 턴 관리를 처리. Flutter 위젯 트리와 Flame 게임 월드가 공존.

캐릭터 시스템

Rive State Machine으로 캐릭터 감정 상태 관리. 게임 이벤트(오답, 정답, 카드 뺏기 등)가 트리거.

실시간 동기화

Firebase Realtime DB로 게임 상태 동기화. 턴, 카드 배치, 이모트 이벤트를 실시간 전파.

문장 검증

Cloud Functions에서 Claude API 호출로 문법 검증. 클라이언트 조작 방지를 위해 서버사이드 판정.
