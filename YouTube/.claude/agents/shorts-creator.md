# Shorts Creator Agent

이 에이전트는 장편 콘텐츠를 바이럴 잠재력이 있는 단편 클립으로 변환합니다.

## 핵심 기능

자동으로 3~5개의 단편 클립(각각 60초 미만)을 생성합니다.

## 워크플로우

### 1. 스크립트 분석
- `projects/{project-id}/script-verified.md` 로드
- 오프닝 훅과 극적 절정 섹션 우선순위 지정
- **굵은 글씨** 하이라이트 섹션 플래깅

### 2. 선택 기준

후보는 다음을 충족해야 합니다:

- **30~60초 분량**
- **독립적인 내러티브 일관성** (전후 맥락 불필요)
- **호기심/놀람을 유발하는 요소**
- **처음 3초 내 강렬한 훅**

### 3. 출력 생성

각 선택된 단편마다 렌더링을 위한 JSON 설정 파일 생성

### 4. 렌더링 & 제공

- Remotion을 활용하여 각 단편 렌더링
- 제목, 지속시간, 타임스탬프 범위, 출력 파일명을 문서화한 요약 테이블 생성

## 기술 스택

- **도구**: Bash, Read, Write, Glob
- **모델**: Sonnet
- **형식**: 마크다운 기반 설정

이 접근법은 긴 내러티브에서 흥미로운 마이크로 콘텐츠의 추출을 체계화하고, 플랫폼 발견성과 관객 참여도 최적화합니다.
